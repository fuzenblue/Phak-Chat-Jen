# 🥗 Phak Chat Jen (ผักชัดเจน)
Phak Chat Jen (ผักชัดเจน) AI-Driven Dynamic Pricing &amp; Hyper-Local Marketplace for Micro-Retailers ชัดเจนเรื่องคุณภาพ ชัดเจนเรื่องราคา ชัดเจนเรื่องพิกัด

## Overview

**Phak Chat Jen (ผักชัดเจน)** คือ Web Application (PWA) ที่ทำหน้าที่เป็น **AI Agent** ช่วยร้านโชห่วยและแผงลอยระบายสินค้าสด (ผัก/ผลไม้) ก่อนเน่าเสีย โดยระบบจะใช้ AI วิเคราะห์ความสดจากรูปถ่ายและคำนวณราคาที่เหมาะสม (Dynamic Pricing) เพื่อจูงใจลูกค้าในละแวกใกล้เคียงผ่านแผนที่ดิจิทัล

โครงการนี้มุ่งเน้นการแก้ปัญหา **Inventory Loss** ของผู้รายย่อย และช่วยให้ผู้บริโภคเข้าถึงวัตถุดิบราคาถูกในช่วงวิกฤตค่าครองชีพ พร้อมลดปัญหา **Food Waste** อย่างยั่งยืน

## ✨ Key Features

### For Sellers - "Snap, Price, Sell"
* **AI Valuation:** วิเคราะห์ความสดของผักจากภาพถ่าย (RGB) ด้วยเทคโนโลยี Computer Vision ที่แม่นยำ
* **Dynamic Pricing Agent:** คำนวณราคาแนะนำอัตโนมัติ โดยอิงจาก *Freshness Score* และ *Shelf Life* (Depreciation Model)
* **One-Click Listing:** ลงขายสินค้าทันที ลดขั้นตอนยุ่งยาก ตอบโจทย์ผู้ใช้งานทุกวัย

### For Buyers - "Find, Navigate, Save"
* **Hyper-Local Discovery:** ค้นหาสินค้าลดราคาในรัศมีใกล้บ้าน (Geo-Spatial Discovery)
* **Real-time Inventory Map:** แสดงพิกัดร้านค้าและสถานะสินค้าบน Google Maps/Longdo Map
* **Deal Navigation:** ระบบนำทางไปยังร้านค้าเพื่อซื้อของ (O2O: Online to Offline)
